Build Command (run inside fika folder):
docker build --no-cache --label modded-fika -t modded-fika .

Docker Commands (run inside server folder):
docker run --pull=never -v $HOME/docker/containers/spt-fika-modded/server:/opt/server -v $HOME/docker/logs:$HOME/docker/logs -p 6969:6969 -p 6970:6970 -p 6971:6971 -it --name modded-fika --log-opt max-size=10m --log-opt max-file=3 modded-fika

docker start modded-fika

docker logs modded-fika -f

docker stop modded-fika

docker update --restart unless-stopped modded-fika  (after 1st run)

docker ps -a

docker rm [container-name/id]

docker images -a

docker rmi [image-name/id]

docker image prune

docker builder prune

Ownership Commands:
sudo chmod -R 775 /home/ubuntu/docker/containers/spt-fika/server/user/profiles
sudo chown -R ubuntu:ubuntu /home/ubuntu/docker/containers/spt-fika/server/user/profiles

sudo chmod -R 775 /home/ubuntu/docker/containers/spt-fika/server/user/mods
sudo chown -R ubuntu:ubuntu /home/ubuntu/docker/containers/spt-fika/server/user/mods

Reboot Command (run inside fika folder):
sudo ./restart_fika.sh
